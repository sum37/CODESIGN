# CODESIGN IDE - 기능 시연 시나리오

## 📌 프로젝트 개요
CODESIGN IDE는 **UI ↔ Code 양방향 편집**을 지원하는 React 디자인 IDE입니다.
Canvas에서 시각적으로 요소를 편집하면 코드가 자동으로 업데이트되고, 코드를 수정하면 Canvas에 즉시 반영됩니다.

---

## 🎬 시연 시나리오

### 시나리오 1: 프로젝트 시작
1. **앱 실행** - CODESIGN IDE 실행
2. **폴더 선택** - "Select Project Folder" 버튼 클릭
3. **ScenarioCode 폴더 선택** - 시나리오 코드가 있는 폴더 선택
4. **IDE 화면 진입** - 3패널 레이아웃 (파일트리 | 캔버스 | 코드에디터)

---

### 시나리오 2: 파일 탐색 및 선택
**사용 파일: `BasicDemo.tsx`**

1. 파일 트리에서 `BasicDemo.tsx` 클릭
2. 캔버스에 컴포넌트 미리보기 렌더링
3. 코드 에디터에 소스 코드 표시

**확인 포인트:**
- ✅ 파일 선택 시 즉시 캔버스와 에디터 동기화
- ✅ TSX/JSX 파일만 캔버스 미리보기 지원

---

### 시나리오 3: 캔버스에서 요소 선택 및 이동
**사용 파일: `BasicDemo.tsx`**

1. 캔버스에서 요소 클릭 → Ghost Box 표시
2. Ghost Box 드래그하여 요소 이동
3. 코드 에디터에서 `left`, `top` 스타일 값 자동 변경 확인

**확인 포인트:**
- ✅ 요소 선택 시 파란 점선 테두리 표시
- ✅ 드래그 완료 시 코드 자동 업데이트
- ✅ position: absolute, left, top 스타일 자동 생성

---

### 시나리오 4: 요소 리사이즈
**사용 파일: `BasicDemo.tsx`**

1. 요소 선택 후 모서리/변 핸들 드래그
2. 요소 크기 변경
3. 코드에서 `width`, `height` 값 자동 변경 확인

**확인 포인트:**
- ✅ 8방향 리사이즈 핸들 (n, s, e, w, ne, nw, se, sw)
- ✅ 최소 크기 20px 제한

---

### 시나리오 5: 도형 추가
**사용 파일: `ShapesDemo.tsx`**

1. 툴바에서 도형 버튼 클릭
2. 도형 종류 선택 (사각형, 원, 삼각형, 별 등)
3. 캔버스에서 드래그하여 도형 생성
4. 코드에 도형 JSX 자동 삽입 확인

**지원 도형:**
- 🟦 사각형 (Rectangle)
- 🟦 둥근 사각형 (Rounded Rectangle)
- 🔷 평행사변형 (Parallelogram)
- ⚪ 원 (Circle)
- 🥚 타원 (Ellipse)
- 🔺 삼각형 (Triangle)
- 💎 마름모 (Diamond)
- ⭐ 별 (Star)
- ⬠ 오각형 (Pentagon)
- ⬡ 육각형 (Hexagon)

---

### 시나리오 6: 텍스트 박스 추가
**사용 파일: `TextDemo.tsx`**

1. 툴바에서 텍스트 버튼 (T) 클릭
2. 캔버스에서 원하는 위치 클릭
3. 텍스트 박스 생성 및 기본 텍스트 표시
4. 더블클릭하여 텍스트 직접 편집

**확인 포인트:**
- ✅ 클릭 위치에 텍스트 박스 생성
- ✅ 더블클릭으로 인라인 편집 모드 진입
- ✅ Enter로 편집 완료, ESC로 취소

---

### 시나리오 7: 텍스트 스타일 편집
**사용 파일: `TextDemo.tsx`**

1. 텍스트 요소 선택
2. 툴바에서 스타일 변경:
   - 폰트 크기 (12px ~ 72px)
   - 폰트 패밀리 (Arial, Georgia, Courier 등)
   - 굵기 (Bold/Normal)
   - 기울임 (Italic/Normal)
   - 색상 선택
   - 정렬 (Left, Center, Right)
3. 코드에 스타일 자동 반영 확인

---

### 시나리오 8: 도형 스타일 편집
**사용 파일: `ShapesDemo.tsx`**

1. 도형 요소 선택
2. 툴바에서 스타일 변경:
   - 배경색 (Fill Color)
   - 테두리 색상 및 두께 (Stroke)
   - 테두리 반경 (Border Radius) - 둥근 사각형
   - 그림자 효과 (Shadow: None/Outer/Inner)
   - 투명도 (Opacity)
3. 코드에 스타일 자동 반영 확인

---

### 시나리오 9: 코드 → 캔버스 동기화
**사용 파일: `BasicDemo.tsx`**

1. 코드 에디터에서 직접 코드 수정
2. `Ctrl+S` (또는 `Cmd+S`) 로 저장
3. 캔버스에 변경사항 즉시 반영 확인

**예시 수정:**
```tsx
// 배경색 변경
backgroundColor: "#f9a8d4"  →  backgroundColor: "#10b981"

// 텍스트 변경
"Hello World"  →  "안녕하세요!"

// 크기 변경
width: "200px"  →  width: "300px"
```

---

### 시나리오 10: 요소 삭제
**사용 파일: `ShapesDemo.tsx`**

1. 요소 선택
2. `Delete` 키 또는 우클릭 → 삭제 메뉴
3. 요소 삭제 및 코드에서 JSX 자동 제거 확인

---

### 시나리오 11: 줌 기능
**사용 파일: 아무 파일**

1. `Ctrl/Cmd + 휠 스크롤` - 줌 인/아웃
2. 줌 레벨 표시 (25% ~ 300%)
3. `Reset` 버튼으로 100%로 복귀
4. 핀치 줌 지원 (터치 디바이스)

---

### 시나리오 12: 요소 잠금
**사용 파일: `ShapesDemo.tsx`**

1. 요소 선택
2. 툴바에서 잠금 버튼 클릭
3. 잠긴 요소는 드래그/리사이즈 불가
4. 잠금 해제하여 다시 편집 가능

---

### 시나리오 13: 새 파일 생성
1. 파일 트리에서 "+ New File" 버튼 클릭
2. `NewComponent.tsx` 파일 자동 생성
3. 기본 React 컴포넌트 템플릿 적용
4. 새 파일에서 작업 시작

---

### 시나리오 14: 복잡한 컴포넌트 편집
**사용 파일: `ComplexDemo.tsx`**

1. 중첩된 요소 구조 확인
2. 자식 요소 개별 선택 및 편집
3. 부모-자식 관계 유지하면서 스타일 변경
4. 코드에서 구조 확인

---

### 시나리오 15: 리사이즈 가능한 패널
1. 파일 트리와 캔버스 사이 경계선 드래그
2. 캔버스와 코드 에디터 사이 경계선 드래그
3. 작업 환경에 맞게 패널 크기 조정

---

## 🔑 핵심 기능 요약

| 기능 | 설명 | 단축키/조작 |
|------|------|------------|
| 요소 선택 | Ghost Box 표시 | 클릭 |
| 요소 이동 | 드래그 앤 드롭 | 마우스 드래그 |
| 요소 리사이즈 | 핸들 드래그 | 모서리/변 드래그 |
| 텍스트 편집 | 인라인 편집 | 더블클릭 |
| 저장 | 코드 저장 및 동기화 | Ctrl/Cmd + S |
| 삭제 | 요소 삭제 | Delete / Backspace |
| 줌 | 캔버스 확대/축소 | Ctrl/Cmd + 휠 |
| 그리기 취소 | 도형 그리기 취소 | ESC |
